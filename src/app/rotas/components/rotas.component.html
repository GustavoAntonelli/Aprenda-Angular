O que são rotas ?

Em um aplicativo de página única, você altera o que o usuário vê, mostrando ou ocultando partes da tela que correspondem a componentes específicos, em vez de ir ao servidor para obter uma nova página. À medida que os usuários executam as tarefas do aplicativo, eles precisam se mover entre as diferentes visualizações que você definiu.

Para controlar a navegação de uma view para a próxima, você usa o Angular Router. O Router permite a navegação interpretando um URL do navegador como uma instrução para alterar a visualização.

Pré-requisitos

Para criar um service, verifique se está seguindo estes pré-requisitos

  1. Instale o Angular CLI com o seguinte comando: npm install g @angular/cli
  2. Veja se ja esta com o projeto criado. Se ainda nao possui um projeto, crie um usando o seguinte comando no prompt de comando: ng new nome-do-projeto, onde nome-do-projeto é o nome da sua aplicação em Angular
  3. Digite o comando: ng serve --open. Seu navegador irá abrir com a pagina do Angular informando que seu projeto está funcionando.
